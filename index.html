Below is a selection of my most recent personal projects. Select any of them for a brief description and media demonstration.

◈Business Analysis, SRS, and Prototype:◈

As a means to explore the business development end of software engineering, I crafted a series of requirements documentation as well as a prototype for a software system that was developed for a local business- Citrus Seafood Restaurant. The reports included: Elicitation Report, Analysis Report, Specification Report, and Validation Report. These were then used as composites to form a Software Requirements Specification, which was serves as the master reference document for the software in development.

◈Yes/No Game:◈

Using artificial intelligence, I created a Wild Thornberry's themed yes/no game, where Nigel Thornberry tries to guess the animal the user is thinking of through a series of yes and no questions. Components: Java, SceneBuilder

◈Production Database:◈

Using Java and an H2 database, I created a production factory production system mockup that allows the user to enter new items into the production line, create 'x' amount of those items, and generate a production log with uniquely generated serial numbers. Initally, the program is pasword locked, and only users with provided credentials can use it. However, once logged in, they can add employees to the system. All these features were implemented using the follwing skills: encapsulation, enumeration, polymorphism, exception handling, lambda expressions, and recursion. Components: Java, SQL, H2, SceneBuilder

◈Retro Pie:◈

This simple Raspberry Pie project has brought me a great deal of fun. Using a Raspberry Pie 3, and a set of online instructions, I was able to create a retro gaming emulation station to replay old favorites and experience classics for the first time. The system is HDMI and ethernet ready, with four USB ports for controllers. Below are a few attached photos of the process. Components: RaspberryPi, RetroPie Software, Various hardware

◈Embeded System Traffic Light Simuation:◈

For my final assembly project, I chose to simulate a traffic light with an interruptive crosswalk button. In essence, the program functions similarly to a traffic light by cycling between red, yellow, and green LED's w/ 3 sec delays until a global interrupt, in the form of a “crosswalk” button, is triggered into completing the current cycle then delaying on the red LED for an extra 10 secs (15 total) before returning to the toggle cycle.

◈Cipherit:◈

As participants in UCF's Knight Hacks 2020 Hackathon, my team and I created a basic encryption tool in a 36 hr period. The tool, called Cipherit, offers various string cryptography, such as: Atbash, Casear, Vignere, and Numeric. As well as a proof of concept form of picture encryption. Components: Java, SceneBuilder

◈Land of Locke:◈

This is a simple game demo I made using MIT's Scratch progam. Unfortunately, because of an embedding error on MIT's end, the game may not load properly. So, below is a video demonstration, and I've included a link to play the actual demo- here. Components: Scratch, Audacity, FL Studios

◈Predictive Text Analysis:◈

Using Python and simple Affinity Analysis, I developed a simple program that when given a word will suggest/predict what the following word may be. More elaborately, the algorithim references a lexicon of texts to scan for instances of a given word to determine the probability will be of the following word. If no word with decent probability confidence is found, the program pads the suggestions with arbitrary filler words, such as "the" or "this".
